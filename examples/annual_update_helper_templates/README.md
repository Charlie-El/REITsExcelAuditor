# 年报更新辅助模板

这个目录是 `年报现金流更新` 功能的用户模板区，所有文件都应保持为空白模板，不放真实业务数据。

## 先用哪张表

当前最推荐主模板是：

```text
年度更新_统一补充大表模板.xlsx
```

这张表尽量把基础信息、OCR/AI 整理后的年度数据、人工补充字段放到一起。字段不需要一次补全，缺少的内容会留空或进入复核清单。

## 模板用途

| 文件 | 用途 |
| --- | --- |
| `年度更新_统一补充大表模板.xlsx` | 推荐主模板，集中维护年度更新所需字段。 |
| `年度更新_标准导入表模板.xlsx` | 旧版标准导入模板，仍可继续使用。 |
| `年度更新_项目别名映射表模板.xlsx` | 当 OCR/人工整理后的项目名和正式表项目名不一致时使用。 |
| `年度更新_基金净资产与折旧摊销参考表模板.xlsx` | 用来对照年报 PDF 自动提取结果。 |
| `年度更新_管理费率辅助表模板.xlsx` | 单独补管理费率、托管费率等。 |
| `年度更新_评估价值与资产性质辅助表模板.xlsx` | 单独补评估价值、资产性质、折现率等。 |
| `年度更新_残值参数辅助表模板.xlsx` | 单独补残值年度、残值参数等。 |
| `年度更新_公告日期辅助表模板.xlsx` | 单独补公告日期。 |

## 推荐放置方式

```text
你的年报工作文件夹\
  02_补充资料\
    年度更新_统一补充大表.xlsx
    年度更新_项目别名映射表.xlsx
    年度更新_基金净资产与折旧摊销参考表.xlsx
```

程序主要按表头识别，不强依赖文件名完全一致。

## 最后提醒

真实填写后的文件请放到你自己的工作文件夹里，不要直接留在模板目录中。
