# examples 目录说明

这个目录只放给用户复制使用的空白模板，不放真实基金数据。

## 这里有什么

- `自动审核补全信息表模板.xlsx`
  给第一个功能 `Excel 自动审核转换` 使用。
- `annual_update_helper_templates\`
  给第二个功能 `年报现金流更新` 使用。

## 自动审核补全信息表

`自动审核补全信息表模板.xlsx` 用于在转换时自动补充 REITs 基础信息，例如 REITs 名称、上市日期、公告日期、开始日期和结束日期。

建议在这些情况下使用：

- 源表缺少 `REITs名称` 或 `REITs简称`。
- 源表缺少 `上市日期`、`公告日期`、`开始日期` 或 `结束日期`。
- 自定义模板中需要 `REITs代码`、`REITs名称`、`INFOPUBLDATE`、`STARTDATE`、`ENDDATE` 等字段，但源表没有。

## 年报更新辅助模板

`annual_update_helper_templates\` 里是年度更新流程使用的空白辅助表。推荐优先使用：

```text
annual_update_helper_templates\年度更新_统一补充大表模板.xlsx
```

如果你希望拆分维护，也可以使用管理费率、评估价值、残值参数、公告日期、项目别名映射、基金净资产与折旧摊销参考表等辅助模板。

## 推荐怎么用

不要直接在这个目录里填写真实数据。建议做法是：

1. 先复制你需要的模板到自己的工作文件夹。
2. 在复制后的文件里填写真实内容。
3. 运行程序时选择你自己的工作文件夹，而不是直接改仓库里的模板。

这样可以避免把真实业务数据混进模板目录，也方便以后统一更新模板。
