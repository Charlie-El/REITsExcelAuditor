# 标准模板说明

本目录保存软件内置转换逻辑使用的标准 Excel 模板。软件会读取这些模板的表头、工作表名称、列宽、字体、边框、筛选范围和数字格式，并把转换后的数据写入对应模板。

## 当前模板

| 模板文件 | 对应输出 |
| --- | --- |
| `01-基础资产估值标准模板.xlsx` | 基础资产估值 |
| `02-高速经营数据标准模板.xlsx` | 高速经营数据 |
| `03-资产负债收入成本标准模板.xlsx` | 资产负债收入成本 |
| `04-产权经营数据标准模板.xlsx` | 产权经营数据 |
| `05-能源经营数据标准模板.xlsx` | 能源经营数据 |

## 更换模板

如果需要替换某一类输出模板，可以按下面步骤操作：

1. 准备新的 `.xlsx` 模板。
2. 确认模板表头是你希望输出的标准字段。
3. 确认模板的字体、列宽、边框、数字格式和筛选范围已经调整好。
4. 将模板文件放到 `standard_templates\` 目录。
5. 打开 `config\table_templates.json`，把对应类型的文件名改成新模板文件名。
6. 重新打开软件后再转换。

## 模板命名

建议使用稳定、清晰的中文标准文件名，例如：

| 类型 | 推荐文件名 |
| --- | --- |
| 第 1 类 | `01-基础资产估值标准模板.xlsx` |
| 第 2 类 | `02-高速经营数据标准模板.xlsx` |
| 第 3 类 | `03-资产负债收入成本标准模板.xlsx` |
| 第 4 类 | `04-产权经营数据标准模板.xlsx` |
| 第 5 类 | `05-能源经营数据标准模板.xlsx` |

文件名不需要包含月份，也不建议使用太长的业务描述。真正决定软件使用哪个模板的是 `config\table_templates.json`。

## 注意事项

- 模板文件必须是 `.xlsx`。
- 不要删除表头行。
- 不要把不需要输出的空白列放入筛选范围。
- 日期、百分比、货币和数字列建议提前在模板里设置好格式。
- 如果模板字段名称变化较大，可以在 `config\field_aliases.json` 中增加字段别名。
- 模板最后一列建议是有效表头，不建议保留空白尾列。
