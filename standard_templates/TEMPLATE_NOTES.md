# 标准模板说明

本目录保存软件内置转换逻辑使用的标准 Excel 模板。软件会读取这些模板的表头、工作表名称、列宽、字体、边框、筛选范围、颜色布局和数字格式，并把转换后的数据写入对应模板。

## 两类模板

### 1. `excel_conversion\`

给第一个功能 `Excel 自动审核转换` 使用的固定模板：

| 模板文件 | 对应输出 |
| --- | --- |
| `01-基础资产估值标准模板.xlsx` | 基础资产估值 |
| `02-高速经营数据标准模板.xlsx` | 高速经营数据 |
| `03-资产负债收入成本标准模板.xlsx` | 资产负债收入成本 |
| `04-产权经营数据标准模板.xlsx` | 产权经营数据 |
| `05-能源经营数据标准模板.xlsx` | 能源经营数据 |

这些模板由 `config\table_templates.json` 指定。更换文件名后，需要同步修改配置。

### 2. `annual_update\`

给第二个功能 `年报现金流更新` 使用的标准格式模板：

| 模板文件 | 对应输出 |
| --- | --- |
| `产权年报提取模板.xlsx` | 产权类年报提取输出 |
| `特许经营权年报提取模板.xlsx` | 特许经营权类年报提取输出 |
| `未来现金流模板.xlsx` | 未来现金流汇总输出 |

这些模板是脱敏骨架模板，只保留格式、颜色、列宽、表头、基础行高和可打开性，不包含真实基金数据。当前年报更新默认优先使用这里的模板作为标准格式来源。

## 更换模板

如果需要替换某一类输出模板，可以按下面步骤操作：

1. 准备新的 `.xlsx` 模板。
2. 确认模板表头是程序能识别的标准字段。
3. 确认字体、列宽、边框、颜色、数字格式和筛选范围已经调整好。
4. 放到对应目录。
5. 1-5 类自动审核模板需要同步修改 `config\table_templates.json`。
6. 年报更新模板请保持文件名不变，或同步修改程序内置模板文件名。
7. 用小样本输出一次，打开 Excel 检查格式和公式。

## 注意事项

- 模板文件必须是 `.xlsx`。
- 不要删除表头行。
- 不要在模板里保留真实基金数据。
- 不要把工作过程输出、已核结果、年报 PDF 或 OCR 图片放进本目录。
- 日期、百分比、货币和数字列建议提前在模板里设置好格式。
- 模板最后一列建议是有效表头，不建议保留空白尾列。
